About This Tool

This tool is designed to assist in the preliminary analysis of Fine Needle Aspiration (FNA) cytopathology images. It accepts a JPG image of an FNA slide and uses a backend machine learning model to predict whether the cells in the image indicate a cancerous or non-cancerous tumor.

How it Works:

  1. Upload: Select a clear JPG image of an FNA cytopathology slide.
  2. Analysis: The image is sent to our secure backend server. A pre-trained deep learning model processes the image features.
  3. Result: The model outputs a prediction: '1' for cancerous, '0' for non-cancerous. This result is then displayed back to you.

Technology:

The frontend is built with HTML, CSS, and JavaScript. The backend employs a well-orchestrated Python-based image processing pipeline and machine learning framework for feature extraction and a pre-trained Logistic Regression model employing KNN and cross-validation for predicting the nature of breast tumors (benign or malignant) from uploaded FNA cytopathology images.
